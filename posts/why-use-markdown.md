# Why Markdown Rocks

- **Simple syntax** (no HTML clutter)
- **Portable** (works everywhere)
- Supports tables:

| Syntax      | Description |
| ----------- | ----------- |
| `#`         | Heading     |
| `**text**` | Bold        |
